Подключение скриптов и загрузка данных

На графике изображены исходные данные - помесячные продажи вин в Австралии, а также периодограмма всего ряда, на которой видно наличие тренда и сильно выраженная сезонность.

library("LSTS")
source("auto_ssa_decomposition.R")
printf <- function(str, res) {
  print(paste(str, res), quote = FALSE)
}
time_series <- wineind
time_series_length <- length(time_series)
plot(time_series, type = "l", main = "Initial time series")

per <- periodogram(time_series)
print(per$plot + ggtitle("Periodogram of the initial time series"))

Оценка ранга сигнала

В следующем фрагменте показано, как можно оценить ранг сигнала. Заметим, что в данном случае он оценился правильно. Однако в общем случае рекомендуется искусственно завышать оценку, если необходимо выделить весь сигнал. Разработанные алгоритмы позволяют, в основном, отсеять лишние компоненты.

Правильность оценки ранга определили по отсутствию в остатке тренда и сезонности, что показывает периодограмма остатка.

window_length <- 48
rank <-
  estimate.rank(
    time_series,
    L = window_length,
    method = "trmat",
    rank_range = 0:20,
    check_stat = FALSE
  )
printf("Estimated rank", rank)
## [1] Estimated rank 13
s <- ssa(time_series, L = window_length)
rec <- reconstruct(s, groups = list(1:rank))

per <- periodogram(residuals(rec))
print(per$plot + ggtitle("Periodogram of the residuals after rank detection"))

Идентификация трендовых и периодических компонент

Задание параметров

Зададим список параметров на основе которых будет проходить идентификация. Выбор порогов основан на проведенных численных экспериментах для модельных данных. Порог низких частот выбран равным половине минимальной частоты сезонности. Установлен более общая модель шума, однако в данном случае результат устойчив по отношению к выбору модели.

method_parameters <-
  list(
    'window_length' = window_length,
    # Длина окна
    'signal_rank' = rank,
    # Оценка ранга сигнала
    'auto_trend_freq' = 1 / 24,
    # Порог низких частот
    'auto_threshold' = 0.5,
    # Порог для вклада низких частот
    'tau_threshold' = 0.05,
    # Порог для меры tau
    'p_0' = 0.03,
    # Порог для меры в идентификации периодик с периодом 2
    'sign_level' = 0.05,
    # Уровень значимости для проверки гипотезы о шуме
    'noise_type' = 'red' # Тип шума
  )

Вызов автоматической идентификации компонент

res <- auto.full.decompose(
  time_series = time_series,
  L = method_parameters$window_length,
  signal_rank = method_parameters$signal_rank,
  sign_level = method_parameters$sign_level,
  tau_threshold = method_parameters$tau_threshold,
  auto_trend_freq = method_parameters$auto_trend_freq,
  auto_threshold = method_parameters$auto_threshold,
  p_0 = method_parameters$p_0,
  noise_type = method_parameters$noise_type,
  check_hypothesis = TRUE
)

Вывод результатов
